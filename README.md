# Научиться учиться
Проектная работа Яндекс практикума.Небольшая web-страница на 11блоков на которых мы отрабатываем пройденный материал.
Для построения этой страницы используеться технология органицации файловой структуры БЭМ-Nested.
Также технологии выравнивания блоков и элементов.
Технология анимации и трансформации объектов.
